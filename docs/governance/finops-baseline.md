# FinOps baseline inicial por tenant

## Objetivo
Registrar costo base mensual estimado por tenant para detectar desviaciones tempranas.

## Campos mínimos
- tenant_id
- entorno (staging/producción)
- costo mensual estimado
- costo real (cuando aplique)
- variación (%)
- observaciones/acciones

## Estado inicial
- Baseline creado (pendiente de carga de datos reales).
